# AI Finance App

Full-stack demo: backend (Express + db) and frontend (React). See backend/ and frontend/ folders. Start backend: `cd backend && npm install && npm start`. Start frontend: `cd frontend && npm install && npm run dev`.

Project Folder Structure:

AI-FINANCE-APP/
│
├── backend/
│   ├── node_modules/
│   ├── .env.example
│   ├── db.json
│   ├── index.js
│   ├── package.json
│   └── README.md
│
└── frontend/
    ├── node_modules/
    ├── public/   (optional)
    ├── src/
    │   ├── auth/
    │   ├── dashboard/
    │   ├── App.jsx
    │   └── main.jsx
    ├── styles.css
    ├── index.html
    ├── package.json
    └── README.md
